# grocery-store
React WebApp using create-react-app

# Tools

- create-react-app cli
- Reactjs-toolkit
- postgresql-typeorm

- ### Start App:
- ### Clone repo,
- ### install:
   #### cd client npm install or yarn
   #### cd server npm install or yarn
   #### cd into main folder npm install or yarn then run:
   #### npm run dev or yarn dev
